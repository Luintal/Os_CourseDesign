# Os_CourseDesign
# 2024年操作系统课程设计

# 课程设计内容

设计一个文件系统，要求实现文件的创建、删除、修改和查询等功能，以及用户的注册和登录功能。用户之间的文件系统互不干扰，文件系统采用两级文件目录结构，第一级为用户账号，第二级为用户账号下的文件系统。为了简便文件系统的设计，可以不实现文件共享、文件系统的安全、管道文件与设备文件等特殊内容。
